# Consejo-Editorial-UAML
Repositorio institucional para la gestión y archivado digital de las publicaciones de las revistas académicas An@lítica y Flectere, editadas por el Comité Editorial de la DCSH del a UAM Lerma. Incluye artículos, procesos editoriales y recursos asociados al fortalecimiento de la divulgación científica en ciencias sociales y humanidades.
